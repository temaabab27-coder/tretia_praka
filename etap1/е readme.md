Практическая работа №1 (Вариант 1)
Ассемблер и интерпретатор для учебной виртуальной машины (УВМ)

Цель работы — реализовать транслятор и исполнитель для простой машины с четырьмя командами: LOAD, READ, WRITE, DIV.

Что сделано
Этап 1: ассемблер. Читает программу из YAML-файла и строит промежуточное представление.
Этап 2: генерация машинного кода. Преобразует промежуточное представление в бинарный файл (.bin). Байты совпадают со спецификацией:

LOAD 893 → 0xA8, 0x6F, 0x00, 0x00
READ 370 → 0x59, 0x2E
WRITE 1014 → 0xC4, 0x7E
DIV 169 → 0x3B, 0x15
Этап 3: интерпретатор. Выполняет бинарный файл, управляет памятью (1024 ячейки) и аккумулятором. Поддерживает вывод дампа памяти в CSV.

Этап 4: безопасное деление. При DIV с нулевым делителем программа печатает ошибку и завершается с кодом 1.

Этап 5: деление векторов. Реализована поэлементная операция:
A = [100, 81, 64]
B = [10, 9, 8]
C[i] = A[i] // B[i] → [10, 9, 8]
Результат записывается в память и сохраняется в memory.csv.

Структура проекта
prac1/

└── README.md — этот файл

Как запустить
Установить зависимости (один раз):
python -m pip install pyyaml
Собрать программу:
python src/assembler.py -i examples/test.yaml -o program.bin --test
Должно вывести 12 байт, как в ТЗ.
Выполнить программу:
python src/interpreter.py -i program.bin -o memory.csv --test
Проверить деление на ноль:
python src/assembler.py -i examples/div_zero.yaml -o div_zero.bin
python src/interpreter.py -i div_zero.bin
Должно вывести: «Ошибка: DIV — деление на ноль в mem[6]» и завершиться.
Запустить деление векторов:
python src/assembler.py -i examples/vector_div.yaml -o vector_div.bin
python src/interpreter.py -i vector_div.bin -o memory.csv
В файле memory.csv будут ячейки 6, 7, 8 со значениями 10, 9, 8.
Команды машины
LOAD const — загрузить константу в аккумулятор
READ addr — прочитать из памяти по адресу в аккумулятор
WRITE addr — записать аккумулятор в память по адресу
DIV addr — разделить аккумулятор на значение из памяти по адресу (целочисленное деление)

Если при DIV в памяти лежит 0 — программа аварийно завершается.
